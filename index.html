Concentration by GothamGazette

A memory or concentration style card game.

This is the code that was developed for Gotham Gazette's Switch game. This release was designed specifically to meet our need for an electricity policy game: each pair of cards represents an measure the city could take to conserve or produce electricity.

As an administrator you can supply an unlimited (not tested past 50) number of available "energy measures" or card options. Players are dealt a game board with 18 of those, split evenly between production and conservation measures.

View a live installation of a game built on this code on Gotham Gazette

Dependencies

Bitweaver (which in turn requires Apache, PHP and one of MySQL, Postgres, Oracle, Firebird, IIS/MS-SQL)
